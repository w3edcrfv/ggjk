# 移动端测试清单

## 测试前准备

- [ ] 确保开发服务器正在运行：`npm run dev`
- [ ] 清除浏览器缓存：Ctrl+Shift+Delete
- [ ] 准备多个测试设备（或使用浏览器模拟器）

---

## 📱 测试项目

### 1. 首页 (http://localhost:3000/)

#### 布局检查
- [ ] 页面标题显示完整（不换行或合理换行）
- [ ] 5个测评对象卡片排列正确
- [ ] 卡片间距合适，不拥挤
- [ ] 测评流程图标可见

#### 交互检查
- [ ] 点击测评对象卡片可以跳转
- [ ] 卡片悬停效果正常（桌面端）
- [ ] 触摸反馈灵敏（移动端）

#### 响应式检查
- [ ] 手机竖屏（375px）：卡片单列显示
- [ ] 平板竖屏（768px）：卡片2列显示
- [ ] 桌面（1024px+）：卡片3列显示

---

### 2. 量表选择页 (选择任一测评对象)

#### 布局检查
- [ ] 面包屑导航显示正确（桌面端显示，移动端隐藏）
- [ ] 返回按钮可见且可点击
- [ ] 测评对象图标和标题显示完整
- [ ] 量表卡片信息完整

#### 交互检查
- [ ] 返回按钮可以返回首页
- [ ] 点击量表卡片可以跳转到答题页

#### 响应式检查
- [ ] 移动端：卡片单列显示
- [ ] 平板：卡片2列显示
- [ ] 桌面：卡片3列显示

---

### 3. 测评答题页

#### 第一步：基本信息
- [ ] 所有表单字段显示完整
- [ ] 输入框可正常输入
- [ ] 单选框/多选框可正常选择
- [ ] 年龄输入验证正常（0-120）
- [ ] "下一步"按钮可点击

#### 第二步：量表答题
- [ ] 题目编号和内容显示完整
- [ ] 单选按钮可正常选择
- [ ] 已选择的题目卡片有绿色标记
- [ ] 进度条正确显示答题进度
- [ ] "提交测评"按钮在全部完成后可点击
- [ ] "上一步"按钮可返回

#### 响应式检查
- [ ] 移动端：表单字段间距合适
- [ ] 移动端：卡片内边距适中
- [ ] 移动端：按钮大小适合触摸（≥44px）
- [ ] 桌面：布局合理，不松散

---

### 4. 结果展示页

#### 布局检查
- [ ] 基本信息显示完整（姓名、性别、年龄、测评时间）
- [ ] 总得分大字体显示
- [ ] 等级标签颜色正确（正常-绿色，轻度/中度-橙色，重度-红色）
- [ ] 结果解释显示完整
- [ ] 柱状图和饼图显示完整
- [ ] 维度得分详情显示完整
- [ ] 专业建议显示完整
- [ ] 注意事项显示完整

#### 交互检查
- [ ] "返回首页"按钮可点击
- [ ] "查看完整报告"按钮可跳转
- [ ] "导出PDF"按钮可下载（需要测试）

#### 响应式检查
- [ ] 移动端：图表高度适中（h-64）
- [ ] 移动端：饼图标签不重叠
- [ ] 移动端：进度条清晰可见
- [ ] 桌面：图表高度适中（h-80）

---

### 5. 报告页 (从结果页点击"查看完整报告")

#### 布局检查
- [ ] 报告标题显示完整
- [ ] 所有章节显示完整
- [ ] 下载按钮可见

#### 交互检查
- [ ] "返回"按钮可返回结果页
- [ ] "导出PDF"按钮可下载
- [ ] "打印"按钮可打开打印窗口

#### 响应式检查
- [ ] 移动端：报告布局紧凑
- [ ] 移动端：字体大小合适
- [ ] 打印预览：布局正确

---

## 🎨 视觉检查

### 颜色和对比度
- [ ] 文字与背景对比度足够（WCAG AA标准）
- [ ] 图表颜色清晰可辨
- [ ] 按钮、卡片等元素颜色一致

### 字体大小
- [ ] 移动端：最小 14px（iOS推荐）
- [ ] 桌面：最小 16px
- [ ] 标题层级分明

### 间距
- [ ] 移动端：元素间距适中（8-16px）
- [ ] 桌面：元素间距舒适（16-24px）
- [ ] 边距统一（padding/margin）

---

## 🚀 性能检查

### 加载速度
- [ ] 首次加载时间 < 3秒（4G网络）
- [ ] 页面切换流畅，无明显卡顿
- [ ] 图片和图标加载正常

### 内存使用
- [ ] 长时间使用无内存泄漏
- [ ] 切换页面内存释放正常
