# 项目管理系统 - 功能介绍文档

## 📋 项目概述

这是一个专为**室内设计与装修行业**打造的全流程项目管理系统，帮助设计公司高效管理从设计到验收的完整项目周期。系统支持多角色协作、权限管理、进度跟踪等核心功能。

---

## 🎯 核心功能模块

### 1. 项目管理看板
**功能描述：** 项目总览与快速访问入口

**主要特性：**
- 📊 项目卡片展示：显示项目状态、客户信息、团队成员
- 🔍 快速筛选：按进行中/已完成状态筛选项目
- 📈 进度可视化：实时显示施工进度百分比
- 🎯 关键日期提醒：交定时间、预计完成日期一目了然
- 👥 团队信息：顾问、设计师信息快速查看
- 🖱️ 一键跳转：点击卡片直接进入项目详情页

**适用角色：** 所有用户

---

### 2. 项目详情管理
**功能描述：** 完整的项目全生命周期管理

#### 2.1 基本信息管理
- 客户名称、项目地址
- 顾问、设计师分配
- 交定时间、确单时间记录

#### 2.2 设计阶段管理
**包含模块：**
- 📝 设计需求记录表：收集客户需求，支持文件/链接上传
- 🎨 细化方案输出：方案文档管理，状态跟踪
- ✅ 确单时间记录：方案确认节点管理

**状态标识：** 已完成 / 进行中 / 待输出

#### 2.3 施工阶段管理
**包含模块：**
- 🏗️ 施工进度表输出：施工计划文档管理
- 📊 当前施工进度：实时进度百分比或完成状态
- 📅 硬装完成日期：预期完成时间跟踪

#### 2.4 定制柜阶段管理
**包含模块：**
- ✍️ 柜体图纸签字：图纸确认时间记录
- 📋 柜体下单日期：订单提交时间
- 🚚 柜体预计到货：物流预期时间
- ✅ 柜体实际到货：实际到货时间记录

#### 2.5 软装阶段管理
**包含模块：**
- 🛋️ 家具灯具窗帘下单日期：软装订单时间
- 📦 家具到货日期：家具物流跟踪
- 💡 灯具安装日期：灯具安装节点
- 🪟 窗帘安装日期：窗帘安装节点

#### 2.6 验收阶段管理
**包含模块：**
- 📅 预计验收日期：计划验收时间
- ✅ 实际验收日期：完成验收时间记录

#### 2.7 项目备注
- 📝 自由文本备注：记录项目特殊要求、客户偏好等重要信息

**适用角色：** 根据权限配置，不同角色可编辑不同阶段

---

### 3. 权限管理系统
**功能描述：** 灵活的角色与权限配置

#### 3.1 角色管理
**支持角色类型：**
- 👑 **管理员**：拥有所有权限，可编辑所有阶段
- 👤 **员工**：可自定义阶段编辑权限

#### 3.2 阶段权限配置
**可配置的权限维度：**
- 🎨 设计阶段权限
- 🏗️ 施工阶段权限
- 📦 定制柜阶段权限
- 🛋️ 软装阶段权限
- ✅ 验收阶段权限

#### 3.3 权限管理功能
- ➕ 从组织架构批量添加员工
- 🔄 动态切换用户角色（管理员/员工）
- ⚙️ 单独配置每个员工的阶段权限
- 🔒 管理员权限锁定，无法修改
- 💾 权限配置保存与应用

**适用角色：** 管理员

---

### 4. 组织架构管理
**功能描述：** 企业组织结构与员工管理

#### 4.1 部门管理
- 📂 多部门支持：设计部、工程部、市场部等
- 🔍 部门快速切换：移动端下拉选择，桌面端侧边栏
- 👥 部门人员统计：实时显示部门人数

#### 4.2 员工管理
- 📇 员工信息展示：姓名、邮箱、职位
- 🔍 员工搜索：快速查找员工
- ☑️ 批量选择：支持多选员工
- ➕ 添加到项目：将员工添加到权限配置

#### 4.3 交互优化
- 📱 移动端适配：底部滑出弹窗，触摸友好
- 💻 桌面端优化：双栏布局，操作高效
- 🎯 已选择提示：实时显示选中人数
- 🗑️ 一键清空：快速清除所有选择

**适用角色：** 管理员

---

### 5. 用户身份切换
**功能描述：** 快速切换不同用户身份进行测试

**主要特性：**
- 🔄 快速切换：下拉选择不同用户
- 👤 用户信息显示：当前用户名称、角色
- 🔐 权限实时生效：切换后立即应用对应权限
- 💾 状态持久化：刷新页面保持登录状态

**适用场景：** 开发测试、权限验证

## 👥 用户角色与权限

### 管理员
**权限范围：**
- ✅ 查看所有项目
- ✅ 编辑所有阶段
- ✅ 配置用户权限
- ✅ 管理组织架构
- ✅ 添加/编辑项目

### 设计师
**典型权限：**
- ✅ 查看分配的项目
- ✅ 编辑设计阶段
- ✅ 编辑定制柜阶段

- ❌ 无验收阶段权限

### 工程师
**典型权限：**
- ✅ 查看分配的项目
- ✅ 编辑施工阶段
- ✅ 编辑定制柜阶段
- ✅ 编辑软装阶段
- ✅ 编辑验收阶段
- ❌ 无设计阶段权限

### 顾问
**典型权限：**
- ✅ 查看分配的项目
- ✅ 编辑设计阶段
- ✅ 编辑施工阶段
- ❌ 其他阶段根据需要配置

## 📊 数据结构

### 项目数据模型
```typescript
interface Project {
  id: number;                          // 项目ID
  clientName: string;                  // 客户名称
  projectAddress: string;              // 项目地址
  consultant: string;                  // 顾问
  designer: string;                    // 设计师
  depositDate: string;                 // 交定时间
  
  // 设计阶段
  designRequirementOutput: string;     // 设计需求状态
  designRequirementFiles: File[];      // 设计需求文件
  detailedPlanOutput: string;          // 细化方案状态
  detailedPlanFiles: File[];           // 细化方案文件
  confirmationDate: string;            // 确单时间
  
  // 施工阶段
  constructionScheduleOutput: string;  // 施工进度表状态
  currentConstructionProgress: string; // 当前施工进度
  constructionCompletionDate: string;  // 硬装完成日期
  
  // 定制柜阶段
  cabinetDrawingSign: string;          // 柜体图纸签字
  cabinetOrderDate: string;            // 柜体下单日期
  cabinetExpectedArrival: string;      // 柜体预计到货
  cabinetActualArrival: string;        // 柜体实际到货
  
  // 软装阶段
  furnitureLightCurtainOrderDate: string; // 家具灯具窗帘下单
  furnitureArrivalDate: string;        // 家具到货日期
  lightingInstallDate: string;         // 灯具安装日期
  curtainInstallDate: string;          // 窗帘安装日期
  
  // 验收阶段
  acceptanceExpectedDate: string;      // 预计验收日期
  acceptanceActualDate: string;        // 实际验收日期
  
  notes: string;                       // 项目备注
}
```

### 用户数据模型
```typescript
interface User {
  id: number;                          // 用户ID
  name: string;                        // 用户名称
  email: string;                       // 邮箱
  role: 'admin' | 'employee';          // 角色
  permissions: {                       // 权限配置
    designStage: boolean;              // 设计阶段权限
    constructionStage: boolean;        // 施工阶段权限
    cabinetStage: boolean;             // 定制柜阶段权限
    softFurnishingStage: boolean;      // 软装阶段权限
    acceptanceStage: boolean;          // 验收阶段权限
  };
}
```

## 🚀 使用场景

### 场景1：设计公司项目管理
**痛点：** 多个项目同时进行，进度难以跟踪
**解决方案：** 
- 项目看板统一管理所有项目
- 状态标签快速识别项目进度
- 关键日期提醒避免延期

### 场景2：团队协作与权限控制
**痛点：** 不同角色需要不同的编辑权限
**解决方案：**
- 灵活的权限配置系统
- 按阶段分配编辑权限
- 管理员统一管理权限

### 场景3：客户信息与进度查询
**痛点：** 客户询问进度时查找困难
**解决方案：**
- 项目详情页完整展示所有信息
- 时间节点清晰记录
- 文件附件集中管理

### 场景4：跨部门协作
**痛点：** 设计、施工、采购等部门信息不同步
**解决方案：**
- 统一的项目信息平台
- 各阶段独立管理
- 实时更新项目状态
